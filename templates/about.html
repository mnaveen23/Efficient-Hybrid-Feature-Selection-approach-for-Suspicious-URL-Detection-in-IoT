About the Project

Project Title:
An Efficient Hybrid Feature Selection Technique Toward Prediction of Suspicious URLs in IoT Environment

In today's interconnected world, the Internet of Things (IoT) is rapidly expanding, making security and privacy more crucial than ever. This project focuses on building a robust prediction system that can accurately identify suspicious URLs, protecting IoT devices from potential cyber threats.

By implementing a hybrid feature selection method, the system intelligently filters and selects the most important characteristics of URLs. This improves prediction accuracy while reducing processing time — making it highly efficient for real-world IoT applications.

Using advanced machine learning techniques such as Gradient Boosting, Random Forest, and Support Vector Machines, the system ensures early detection of malicious URLs, ultimately strengthening the security shield for IoT networks.

Key Features:

  • Hybrid feature selection for optimized prediction.
  • Lightweight and efficient, ideal for IoT devices.
  • High accuracy detection of suspicious URLs.
  • User-friendly web interface for URL testing and results visualization.
